# foxybros
A platform game created using Godot

##About the Project

The game demonstrates core platformer features such as player movement (running and jumping), basic enemy behavior, and item collection. The repository contains scenes, scripts, and assets organized into folders for easy editing and expansion.

## Requirements

Godot Engine (latest stable version recommended)

## How to Run

Clone the repository:
git clone https://github.com/CodingQuests/Platformer-game.git

Open Godot Engine.

Click “Import” and select the project folder.

Open the main scene and press Play.

## Project Structure

Player – Player character scene and scripts

Frog – Enemy example

Collectables – Items the player can pick up

Global – Shared scripts

Main scene files – Game levels and entry point
